# Nộp báo cáo Lab 3 — quy ước chính thức

Tham chiếu: [SCORING.md](../SCORING.md), [EVALUATION.md](../EVALUATION.md), template trong từng thư mục.

## Báo cáo nhóm

| | |
|---|---|
| **Thư mục** | `report/group_report/` |
| **Template** | `TEMPLATE_GROUP_REPORT.md` |
| **File nộp** | `GROUP_REPORT_[TEAM_NAME].md` → **`GROUP_REPORT_PriceCheck.md`** |
| **Cấu trúc** | Đúng 6 mục template (§1–§6), không thêm mục ngoài template |

## Báo cáo cá nhân

| | |
|---|---|
| **Thư mục** | `report/individual_reports/` |
| **Template** | `TEMPLATE_INDIVIDUAL_REPORT.md` |
| **File nộp (SCORING)** | **`individual_report.md`** — đúng tên trong [SCORING.md](../SCORING.md) §2 |
| **Sinh viên** | Vương Sỹ Hạnh — `2A202600722` |
| **Cấu trúc** | Đúng 4 mục: I–IV (không thêm mục V; Agent v2 ghi trong I/II và báo cáo nhóm §5) |

> Ghi chú: Template ghi thêm `REPORT_[YOUR_NAME].md` ở cuối file — **ưu tiên SCORING** (`individual_report.md`) khi nộp bài.
